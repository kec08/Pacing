1. 프로젝트 개요

1.1 앱 소개
Pacing은 러닝 중 근처에 있는 사람과 Apple Music 플레이리스트를 실시간으로 동기화하여 같이 들을 수 있는 iOS 러닝 앱입니다. 지도 위에서 주변 러너를 발견하고, 그 사람이 듣는 음악을 확인한 뒤 '같이 듣기'로 연결되는 새로운 러닝 경험을 제공합니다.

1.2 핵심 가치
	•	혼자 달리지만 혼자가 아닌 느낌 — 음악으로 연결되는 러너 커뮤니티
	•	같은 비트, 같은 페이스 — 음악 싱크를 통한 간접적 동반 러닝
	•	발견의 즐거움 — 지도 위 낯선 러너의 플레이리스트 탐색

1.3 타깃 사용자
	•	Apple Music 구독 중인 20~30대 러너
	•	혼자 러닝하지만 음악으로 동기부여가 필요한 사용자
	•	새로운 음악을 발견하고 싶은 러너

4. 화면 구성 및 플로우

4.1 홈 탭
화면 구성
	•	상단: 인사 텍스트 + 오늘 날짜/날씨
	•	이번 주 러닝 통계 카드 — 총 거리 / 총 시간 / 평균 페이스
	•	최근 러닝 기록 리스트 — 날짜, 거리, 시간, 경로 썸네일
	•	최근 같이 들은 사람 섹션 — 아바타 + 닉네임 + 함께 들은 곡

데이터 소스
	•	Firestore runHistory 컬렉션에서 최근 7일 기록 조회
	•	listenSessions 컬렉션에서 최근 같이 들은 세션 조회

4.2 러닝 탭
기본 화면 구성
	•	전체 화면 MapKit 지도
	•	내 위치 중심 표시 (초록 펄스 링)
	•	주변 러너 핀 — 아바타 + 현재 듣는 곡 버블 실시간 표시
	•	화면 중앙 하단: 대형 시작 버튼
	•	시작 버튼 좌우: 음악 버튼 / 주변 찾기 버튼

시작 버튼 동작
	•	러닝 시작: CoreLocation 고정밀 수집 시작 → 지도에 Polyline 실시간 드로잉 → 상단에 거리/시간 실시간 표시
	•	러닝 중 시작 버튼: 일시정지 버튼으로 전환
	•	종료: 거리/시간/경로 요약 화면 → Firestore 저장

음악 버튼 (Bottom Sheet)
	•	현재 재생 중인 곡 — 앨범아트 / 곡명 / 아티스트
	•	재생/일시정지, 이전/다음 곡 컨트롤
	•	MusicKit ApplicationMusicPlayer 연동

주변 찾기 버튼 (Bottom Sheet)
	•	반경 설정 세그먼트: 300m / 500m / 1km
	•	러너 카드: 아바타 | 닉네임 | 현재 곡명 | 거리 | 같이 듣기 버튼
	•	같이 듣기 버튼 탭 → 연결 요청 전송
	•	상대방 수락 → 싱크 세션 시작 + 상단 배너 표시

같이 듣기 싱크 플로우
	•	1단계: 호스트가 '같이 듣기' 요청 → Realtime DB 세션 생성
	•	2단계: 게스트에게 알림 → 수락/거절
	•	3단계: 수락 시 곡 ID + 재생 위치 + 타임스탬프 전송
	•	4단계: 게스트 기기에서 동일 곡 동일 위치로 재생 시작
	•	5단계: 재생/일시정지/곡 변경 이벤트 실시간 브로드캐스트
	•	6단계: 세션 종료 시 각자 독립 재생 복귀
※ 싱크 오차: 네트워크 레이턴시로 인해 100~300ms 내외 허용 (MVP 기준)

4.3 마이 탭
신체 정보 섹션
	•	프로필 이미지 / 닉네임 / 가입일
	•	키 / 체중 / 나이 / 성별 입력 및 수정
	•	HealthKit 연동 여부 설정 (선택)

활동 통계 섹션
	•	이번 달 총 러닝 거리 / 횟수 / 평균 페이스
	•	월별 러닝 그래프 (막대 차트)
	•	최고 기록 — 최장 거리 / 최고 페이스

최근 활동 섹션
	•	최근 러닝 기록 리스트 (날짜 / 거리 / 시간)
	•	기록 탭 → 상세 화면 (경로 지도 + 구간별 페이스)

설정 섹션
	•	탐색 반경 기본값 설정
	•	Apple Music 연동 상태 확인
	•	알림 설정 (같이 듣기 요청 알림)
	•	로그아웃 / 계정 탈퇴


5. 주요 사용자 플로우

5.1 온보딩 플로우
	•	앱 최초 실행 → Apple 로그인 (Firebase Auth)
	•	위치 권한 요청 (항상 허용 권장)
	•	Apple Music 권한 요청 (MusicKit)
	•	닉네임 + 신체 정보 입력 → 메인 화면 진입

5.2 러닝 시작 플로우
	•	러닝 탭 진입 → 지도 로드, 주변 러너 핀 표시
	•	중앙 시작 버튼 탭
	•	CoreLocation 고정밀 수집 시작
	•	Realtime DB에 내 위치 + 현재 곡 정보 브로드캐스트 (5초 간격)
	•	지도에 이동 경로 Polyline 실시간 드로잉
	•	러닝 종료 → 요약 화면 → Firestore 저장 → 홈 탭에 반영

5.3 같이 듣기 플로우
	•	주변 찾기 버튼 → Bottom Sheet 오픈
	•	원하는 러너의 '같이 듣기' 버튼 탭
	•	상대방 수락 → 싱크 세션 시작
	•	화면 상단에 '지금 함께 듣는 중: [닉네임]' 배너 표시
	•	한쪽이 종료하면 세션 해제, 각자 독립 재생
