# issue-07 GPS 스파이크로 페이스 순간 튀는 현상

## 발생 feat
feat #09 러닝 측정 개선

## 현상
- 러닝 중 페이스가 갑자기 비정상적인 값(매우 빠르거나 느리게)으로 튀는 현상

## 원인
1. GPS 신호 불안정 시 실제 이동하지 않았음에도 순간 수십 미터 점프 발생
2. 순간 페이스(두 GPS 포인트 기반)를 그대로 표시해 노이즈 증폭

## 수정
1. **GPS 스파이크 필터**: 이동 속도 10 m/s(36 km/h) 초과 시 해당 업데이트 무시
2. **페이스 스무딩**: 최근 5개 샘플 이동평균으로 표시값 안정화
